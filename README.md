#  Ecommerce Automation Framework

## Project Overview
This project is an automation testing framework developed using Python, selenium, pytest and Page object model (POM).


## Tools & Technologies
- Python
- Selenium Webdriver
- Pytest
- Pytest HTML reports
- WebDriver Manager

## Framework Features
- Page Object Model (POM)
- Reusable components
- HTML reports
- API testing support
- Cross-browser support

## Project Structure

```bash
pages/
tests/
api_tests/
reports/
conftest.py
requirements.txt
```

## Run UI Tests

```bash
pytest tests/ --html=reports/report.html
```

## Run API tests

```bash
pytest api_tests/
```
